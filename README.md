# Road Accident Risk Prediction (EDA + XGBoost)

This project focuses on analyzing road accident data to understand
key risk factors and build a machine learning model to predict
accident risk levels.

## Problem Statement
Road accidents are a major safety concern. The goal of this project
is to explore accident-related data, identify important patterns,
and build a predictive model that can help assess accident risk.

## Dataset
The dataset is sourced from Kaggle.
It contains multiple features related to road conditions, traffic,
and environmental factors.

> Note: The dataset is not uploaded here due to Kaggle data policies.

## Exploratory Data Analysis (EDA)
In this notebook, I performed:
- Data cleaning and preprocessing
- Missing value analysis
- Feature distributions and relationships
- Correlation analysis
- Identification of important variables

EDA helped in understanding how different factors contribute to
road accident risk.

## Modeling Approach
- Model used: XGBoost
- Data splitting and validation strategy applied
- Model performance evaluated using appropriate metrics

The focus was on building a robust and interpretable model rather
than only maximizing accuracy.

## Key Learnings
- Importance of EDA before modeling
- Feature impact on accident risk
- Handling real-world data issues
- Practical use of XGBoost for tabular data

## Future Improvements
- Hyperparameter tuning
- Feature engineering improvements
- Trying additional models and ensembling
- Better validation strategies
